# API SCRIPT
- Aquesta aplicació va de l'utilització de SharedPreferences per guardar usuaris mitjançant un Login i Register, i l'utilització d'una api en un google sheets mitjançant AppsScript
- Les dades que hem escollit són d'informació de portàtils
- Fet per Marcel Roquet i Pau Solé
